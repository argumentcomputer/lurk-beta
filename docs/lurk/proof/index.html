Module lurk::proof

source ·
Expand description

This module offers a connection the the backend proving engine of Lurk. Abstracted behind the Prover and Verifier traits, the proving engine has two instantiations:

  • the Groth16/SnarkPack proving system, implemented in the groth16 module
  • the Nova proving system, implemented in the nova module.
