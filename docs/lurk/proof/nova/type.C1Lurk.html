pub type C1Lurk<'a, F, C> = MultiFrame<'a, F, C>;
Expand description

Type alias for a MultiFrame with S1 field elements. This uses the <::G1 as Group>::Scalar type for the G1 scalar field elements to reflect it this should not be used outside the Nova context
