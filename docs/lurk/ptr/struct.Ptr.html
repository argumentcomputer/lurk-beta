§tag: ExprTag

An expression tag

The underlying pointer, which can be null, opaque, or an index

§_f: PhantomData<F>

PhantomData is needed to consume the F: LurkField parameter, since we want to pin our Ptr to a specific field (even though we don’t actually use it)
