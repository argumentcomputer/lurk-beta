Expand description

CoCircuit is a trait that represents a generalized interface for coprocessors. Coprocessors augment the Lurk circuit and evaluation with additional built-in functionality. This trait generalizes over functionality needed in the circuit, the sibling Coprocessor trait, generalizes over functionality needed in the evaluator.

The trait is implemented by concrete coprocessor types, such as DumbCoprocessor.
