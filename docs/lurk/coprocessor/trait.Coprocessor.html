Expand description

Coprocessor is a trait that represents a generalized interface for coprocessors. Coprocessors augment the Lurk circuit and evaluation with additional built-in functionality. This trait generalizes over functionality needed in the evaluator, the sibling CoCircuit trait, generalizes over functionality needed in the circuit.

The trait is implemented by concrete coprocessor types, such as DummyCoprocessor.

Pattern

We use a type class (trait) pattern to provide extensibility for the coprocessor implementations. The pattern involves:

  • A trait crate::coprocessor::Coprocessor, which defines the methods and behavior for all coprocessors.
  • An enum such as crate::eval::lang::Coproc, which “closes” the hierarchy of possible coprocessor implementations we want to instantiate at a particular point in the code.
