Lurk DSL

We would like to be able to construct data in a natural way from Rust. This will essentially allow bypassing the string-based reader. The Lurk AST is just Lurk data. This module provides a syntax allowing for natural construction of Lurk data embedded in Rust programs. Even if it has no other benefit, this simplifies code formatting and inline editing of embedded Lurk.

Macros

  • let_store
    Binds a mutable Store to the variable s_, which is (somewhat unfortunately) hard-coded into the lurk! macro. Users should therefore avoid using s_ for other purposes, and will need to use s_ directly when manipulating the Store. The name s_ was chosen to walk the line between obscurity and brevity/clarity. Accidental collisions are undesirable, but so is an awkward or unwieldy name.
